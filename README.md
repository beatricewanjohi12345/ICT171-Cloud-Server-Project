# Project Overview

This project demonstrates the deployment and configuration of a cloud-hosted Linux web server using Microsoft Azure Infrastructure as a Service (IaaS).

The website was manually configured using Ubuntu Linux, SSH remote administration, DNS configuration, SSL/TLS implementation and NGINX web server technologies.

The project also demonstrates GitHub technical documentation and cloud deployment skills.
