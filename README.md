## Project Overview
This project is a cloud-hosted personal portfolio website deployed using Microsoft Azure Infrastructure as a Service (IaaS).

The server was manually configured using SSH access on an Ubuntu Linux virtual machine. NGINX was used as the web server to host the website publicly through a custom domain.
